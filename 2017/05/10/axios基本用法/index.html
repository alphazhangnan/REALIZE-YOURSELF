axios基本用法

发表于 2017-05-10 | 阅读次数

之前做VUE的时候也是了解到，到了VUE2.0，官方HTTP请求就从vue resource改为axios了，那么，我在项目中也用到了axios，下面简单说下axios的基本用法。
Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。
特点

  • 从浏览器中创建 XMLHttpRequests
  • 从 node.js 创建 http 请求
  • 支持 Promise API
  • 拦截请求和响应
  • 转换请求数据和响应数据
  • 取消请求
  • 自动转换 JSON 数据
  • 客户端支持防御 XSRF
